# 📖 Portfolio Website

A creative, interactive portfolio website built with a **flipbook / book-turning animation** concept. Each page flip reveals a new section — giving recruiters and visitors a unique, memorable browsing experience.

## ✨ Features

- 📄 **Flipbook Animation** — Smooth CSS page-turn transitions between sections
- 👤 **Profile Page** — Introduction, social links (LinkedIn, GitHub, LeetCode, CodeChef, Codeforces)
- 🎓 **Education** — Academic background with timeline layout
- 🚀 **Projects** — Detailed project descriptions with tech stacks
- 🛠️ **Skills** — Categorized skills (Languages, Frontend, Backend, Databases, Tools)
- 💼 **Services** — Overview of offered development services
- 📬 **Contact Form** — Built-in contact section

## 🚀 Getting Started

### Prerequisites
Just a modern web browser — no frameworks or build tools required.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Apurva231/apurva-portfolio.git

# Navigate into the project
cd apurva-portfolio

# Open in browser
open index.html
# or just double-click index.html
```
